# Introduction

A suggested learning order may be：

  • Learn highly frequency used module: Define hyperparameter(Params)、Record variable(Meter)、Log(Logger)、Reshape your dataloader(DataBundler) and their aggregation Trainer.
  • Learn how to manage/analyse your experiment by Config and Experiment
  • Learn how to simple manage random seed by RndManager and to create your dataset elegantly by DatasetBuilder

After learning above contents, you can view Cookbook to learn the use of tempelet code and other details.
