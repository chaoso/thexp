# Format output

You can format output easily by using Meter:

from thexp import Meter
m = Meter()
m.a = 0.236
m.b = 3.236
m.c = 0.23612312
m.percent(m.a_)
m.int('b')
m.float(m.c_,2)
print(m)

m.a_ and string 'a'

In Meter ,[name] with underline is equal to the variable name without its underline, e.g., m.a_ == "a", it's just a trick for autocomplete.

# Calculate average

You can contain an average value of meter by using AvgMeter:

from thexp import Meter,AvgMeter

am = AvgMeter()
for j in range(5):
    for i in range(100):
        m = Meter()
        m.percent(m.c_)
        m.a = 1
        m.b = "2"
        m.c = torch.rand(1)[0]

        m.c1 = torch.rand(1)
        m.c2 = torch.rand(2)
        m.c3 = torch.rand(4, 4)
        m.d = [4]
        m.e = {5: "6"}
        # print(m)
        am.update(m)
    print(am)

what will be averaged

AvgMeter only calculate scalar and float variables by default, but you can call average(name) to tell AvgMeter that name need to be calculated as well.







 
 












avg = AvgMeter()

avg.average(avg.a_)
avg.a = 1
avg.a = 2
avg.a = 3
assert avg.a.avg == 2  # AvgItem object
assert avg.a.item == 3

avg.b = 1
avg.b = 2
assert avg.b == 2  # int object

avg.average(avg.c_)
x, y = torch.tensor([1, 2, 3]), torch.tensor([4, 5, 6])
avg.c = x
avg.c = y
assert (avg.c.avg == (x + y) / 2).all()
assert (avg.c.item == y).all()

# Get last updated value or average value

Variables which need to be averaged will be wrapped in an AvgItem in AvgMeter:

from thexp.base_classes.trickitems import AvgItem

from thexp import AvgMeter

avg = AvgMeter()

avg.a = 2.0
avg.a = 4.0
assert isinstance(avg.a, AvgItem)
assert avg.a.avg == 3
assert avg.a.item == 4

avg.b = 5
assert isinstance(avg.b, int)

If you want to get average value or last value, you won't need to judge if the variable is an AvgItem, AvgMeter can 'try' to get it:

avg = AvgMeter()

avg.average(avg.a_)
avg.a = 1
avg.a = 2
avg.a = 3
assert avg.a.avg == 2  # AvgItem object
assert avg.a.item == 3

avg.b = 1
avg.b = 2
assert avg.b == 2  # int object

# try to get last updated value
assert avg.meter.a == 3
assert avg.meter.b == 2

# try to get average value, if can't, get the value itself
assert avg.avg.a == 2
assert avg.avg.b == 2
