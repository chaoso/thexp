# Auto Device

It is the original idea of DataBundler: wrap Dataloader, and automatically assign devices for data.

bundler = DataBundler().add(dataloader_a).cycle(dataloader_b).zip_mode().to(device)


for ... in bundler:
    assert data.device == device # do not need data.to(device)

# Chain and Zip

DataBundler provided two functions to bundler Dataloader: chain and zip

DatasetBuilder support chain call as well.

# chain

Here is an original implement to chain DataLoader one by one:

from itertools import chain
for (xs,ys) in chain(dataloader_a,dataloader_b):
    ...
1
2
3

And not you can also implement it be DataBundler:

from thexp import DataBundler
bundler = DataBundler().add(dataloader_a).add(dataloader_b)

for (xs,ys) in bundler:
    ...

# zip

Here is an original implement to zip different DataLoader:

for (xsa, ysa), (xsb, ysb) in zip(dataloader_a,dataloader_b):
    ...
1
2

And here is a DataBundler version:

from thexp import DataBundler
bundler = DataBundler().add(dataloader_a).add(dataloader_b).zip_mode()

for (xsa, ysa), (xsb, ysb) in bundler:
    ...

# cycle zip

If two Dataloader have different length, zip will be consumed if the shorter one is consumed. So you may need to repeat the shorter dataloader until the longer one is consumed.

from itertools import cycle
for (xsa, ysa), (xsb, ysb) in zip(dataloader_a,cycle(dataloader_b)):
    ...

bundler = DataBundler().add(dataloader_a).cycle(dataloader_b).zip_mode()
for (xsa, ysa), (xsb, ysb) in bundler:
    ...
