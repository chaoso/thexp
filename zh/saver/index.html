# 模型保存：Saver

from thexp import Saver
1

该模块用于保存模型，定义了三种类型的保存方式：checkpoint、keypoint、model。这三种类型能够保存的内容和 torch 覆盖的范围一样（而 torch 的实现实际上通过 pickle 实现，因此能够用 pickle 保存的都可以通过该模块保存）

  • checkpoint：最通用的形式，受 Saver 中的 max_to_keep 参数控制，超出数量的 checkpoint 会根据保存时间顺序删除最早保存的checkpoint，命名形式为 {step}.ckpt。
  • keypoint：不受 max_to_keep 参数影响的 checkpoint ，命名形式为 keep.{step}.ckpt。
  • model：不受 max_to_keep 参数影响的 checkpoint，但命名形式为 model.{step}.pth。
from thexp import Saver

saver = Saver("./sav",max_to_keep=3)
for i in range(10):
    saver.save_keypoint(i,{"a":i},{"b":i})
for i in range(10):
    saver.save_checkpoint(i,{"a":i},{"b":i})
for i in range(10):
    saver.save_model(i,{"a":i},{"b":i})

print(saver.find_keypoints())
print(saver.find_checkpoints())
print(saver.find_models())
