# 继承

你也可以通过继承的方式来实现该类（注意：必须要调用父类的初始化方法）：

from thexp import Params
class MyParams(Params):
    def __init__(self):
        super().__init__() # 父类的初始化方法是必需的
        self.batch_size = 50
        self.topk = (1,2,3,4)
        self.optim = dict(
            lr=0.009,
            moment=0.9
        )

params = MyParams()
print(params)

# 接收命令行参数

在大部分深度学习代码中，接受命令行参数的代码都是非常繁琐的，在pytorch框架中，大多是使用argparse；在tf框架中，大多是使用absl.flags，而实际上，对于深度学习的研究来说，简单使用，所见即所得才是更好的选择。

而通过Google提供的 Fire库，Params可以直接接受命令行参数，免去了各种声明的过程。

如新建一个文件命名为ap.py，你可以用以下三行代码建立一个接受命令行参数的程序。



 

from thexp import Params
params = Params()
params.from_args()

# 限制域

有一些参数可能有显式的声明值域的需求，因此Params中提供了两个方法choice和arange来帮助完成值域的声明：

from thexp import Params
params = Params()
params.choice("dataset","cifar10","cifar100","svhn")
params.arange("thresh",5,0,20)
1
2
3
4

此时，如果为相应的值分配了不在值域范围内的属性，Params会抛出异常。该特性用于更早更快的报出错误，避免因在试验过程中发现这种低级错误而浪费时间

# 关联绑定

params.bind('dataset','cifar10','arch','cnn13')
params.bind('arch','simplenet','arch_param',dict(feature=128))
params.bind('arch','cnn13','arch_param',dict(feature=256))

params.dataset = 'cifar10'

print(params.arch_param)

# Grid Search

Params中添加了Grid Search方法，可以很简单的生成：

iter_dataset = ['cifar10', 'cifar100']
iter_lr = [0.3, 0.1, 0.03, 0.01]
for i, subp in enumerate(params.grid_search('dataset', iter_dataset)):
    for j, subpp in enumerate(subp.grid_search('optim.args.lr', iter_lr)):
        assert subpp.dataset == iter_dataset[i]
        assert subpp.optim.args.lr == iter_lr[j]

# 创建Schedule

Schedule 主要用于动态变化的参数，如学习率调整，损失权重调整，详细的介绍可以参考 Schedule 一节，这里主要介绍通过类内额外添加的类变量 SCHE 来直接定义 Schedule 的方法：

from thexp import Params

params = Params()
params.lr_sche = params.SCHE.Cos(left=0, right=params.epoch, start=0.1, end=1e-5)

# 扩展可被序列化的类

OptimParams 和 Schedule 均支持Params的序列化和反序列化，这是因为其类都继承自 attr，如果你存在想通过Params序列化的类，也可以通过继承 attr 来简单快速的实现。

from thexp.base_classes import attr
from thexp import BaseParams


class MyAttr(attr):
    def __init__(self) -> None:
        super().__init__()
        self.a = 1
        self.b = 2


a = BaseParams()
a.myattr = MyAttr()
a.to_json('tmp.json')

b = BaseParams()
b.from_json('tmp.json')

assert isinstance(b.myattr, MyAttr)
