# 序言

您可以按照如下的顺序对本框架的功能进行学习和使用：

  • 了解使用频率最高的用于训练流程的 超参数声明 (Params) 、变量记录 (Meter) 、日志输出 (Logger) 、数据集整合 (DataBundler) 和将其整合在一起的 训练流程抽象 (Trainer)
  • 了解用于对实验进行分析的 全局配置 (Config) 和 实验管理 (Experiment)
  • 了解面向部份需求的 随机种子管理 (RndManager)、数据集形式构建 (DatasetBuilder)

在对这些内容都有所了解后，您可以通过 Cookbook 查看关于 实验模板 的使用和其他 细节
