# 日志输出：Logger

from thexp import Logger
1

该模块用于日志输出，可以配合Meter类，也可以单独使用，支持多种日志级别的输出，同时添加了在同一行多次输出的支持。

和其他用于日志输出的框架相比，本框架最大的区别在于，当你需要在同一行内输出时，即使字符串长度超过屏幕长度，也不会导致换行，而是会省略多出来的字符串。

# 基本使用

import time
from thexp import Logger,Meter,Params
logger = Logger()

meter = Meter()
meter.a = 3.13524635465
meter.b = "long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;long text;"

for i in range(10):
    logger.inline("inline examples",meter)
    time.sleep(0.2)
logger.info(1,2,3,{4:4})
logger.info(meter)

for i in range(10):
    logger.raw(i,inline=True,append=True) 
    time.sleep(0.2)

在显示屏中运行时你会看到中间一段循环会一直在同一行打印，超过一行的部份会被暂时截断。

线程安全

Logger设计时没有考虑到线程安全，作者暂时没有多进程/并行训练经验，暂时无法对这多线程/进程日志输出进行扩展。

# 添加输出目录

可以添加任意多个输出目录，参数为文件夹位置，日志名称会根据日期时间自动命名，后缀为 .log

logger.add_log_dir("./")
