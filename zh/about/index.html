# 关于其他开源项目

关于该文档的开发必须要感谢众多的开源项目。以下这些开源项目极大的提升了我的开发效率：

  • vuepress-personal，该项目提供了一个完整的基于 vuepress 的项目结构，让我只需要专注于文档本身（Markdown）的写作即可。
  • fitlog，fitlog是一款在深度学习训练中用于辅助用户记录日志和管理代码的工具，该工具中使用 git 的行为和对前端的实现给了我很多灵感，让我最终能够很好的完成试验管理部份的逻辑。

# 关于项目本身

首先要声明的一点是，本项目实际上对于初学者（这包括机器学习初学者和Python初学者）来说不是很友好，因为该框架许多训练流程变成了黑箱子，很多变量也是隐蔽的，很多类重写了许多魔法函数。甚至会有部份代码的表现和初学者的认知大相径庭的可能。因此本项目可能更适合有过几个月甚至一两年编程经验，且对深度学习流程较为熟悉，复现过一些代码，对重复逻辑的地方有了一些认知，体会到一些痛点的人。

另外本项目的开发过程中，慢慢的遵循了 约定优于配置 这一软件范式（当然，这也带来了几次小重构和大重构），这可以很大程度上减轻使用者的学习和使用负担：框架的功能和要求有很多，但你完全不去用也可以，胡乱用也可以，按照约定使用也可以。然而，当你按照框架的约定使用的越多，那么框架能够带给你的支持也就越多，你能够节省的代码和时间也就越多。

目前该项目的整体已经成型，剩下的就是不断的在使用中进行更精细化的补充了。大版本号如无意外不会再进行更新，小版本号定期维护一次。
