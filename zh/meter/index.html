# 输出格式控制

Meter可以很方便的控制输出格式，目前基于需求，主要添加了百分比、浮点精度和整数这三种格式化方法，如下：

from thexp import Meter
m = Meter()
m.a = 0.236
m.b = 3.236
m.c = 0.23612312
m.percent(m.a_)
m.int('b')
m.float(m.c_,2)
print(m)

m.a_ 与 'a'

在 meter 中，变量名+下划线实际等价于去掉了下划线的变量名对应的字符串，也即 m.a_ == "a"，在上面的代码中，这两种形式等价，这种实现方式主要用于IDE的自动补全，在变量后添加一个下划线即可以表示该变量名自身。

# 平均值

在很多时候都需要对输出值取平均，使得观察到的结果比较稳定。这由 thexp.frame.meter.AvgMeter 实现。

from thexp import Meter,AvgMeter

am = AvgMeter()
for j in range(5):
    for i in range(100):
        m = Meter()
        m.percent(m.c_)
        m.a = 1
        m.b = "2"
        m.c = torch.rand(1)[0]

        m.c1 = torch.rand(1)
        m.c2 = torch.rand(2)
        m.c3 = torch.rand(4, 4)
        m.d = [4]
        m.e = {5: "6"}
        # print(m)
        am.update(m)
    print(am)

计算平均值的类型

目前考虑到需求，平均值只计算标量、浮点类型的值，如果整数型值或向量需要计算平均，请将其转换为浮点数。或调用其内部的 average(name) 方法







 
 












avg = AvgMeter()

avg.average(avg.a_)
avg.a = 1
avg.a = 2
avg.a = 3
assert avg.a.avg == 2  # AvgItem object
assert avg.a.item == 3

avg.b = 1
avg.b = 2
assert avg.b == 2  # int object

avg.average(avg.c_)
x, y = torch.tensor([1, 2, 3]), torch.tensor([4, 5, 6])
avg.c = x
avg.c = y
assert (avg.c.avg == (x + y) / 2).all()
assert (avg.c.item == y).all()

# 获取最后更新的值或者平均值

由于是否存储平均值对不同类型规则不同，所以从 AvgMeter 中获取值时，往往需要对其进行一次判断来确认，为了避免繁琐的确认，AvgMeter 提供了 meter 变量和 avg 变量，用来尽量的选取最后更新的值或者平均值。

avg = AvgMeter()

avg.average(avg.a_)
avg.a = 1
avg.a = 2
avg.a = 3
assert avg.a.avg == 2  # AvgItem object
assert avg.a.item == 3

avg.b = 1
avg.b = 2
assert avg.b == 2  # int object

# 对于 meter 来说，尽量的获取最后更新的值
assert avg.meter.a == 3
assert avg.meter.b == 2

# 对于 avg 来说，尽量获取平均值，如果获取不了，则获取最后更新的值
assert avg.avg.a == 2
assert avg.avg.b == 2
