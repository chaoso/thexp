# Schedule

Schedule are used to scale learning rate or weight loss, each subclass of Schedule is a callable class(override __call__ function) that provides series values.

from thexp.calculate import schedule

line_sche = schedule.LinearSchedule(left=0, right=1, start=0, end=1)
line_sche.plot()

print(line_sche(0.75))

# ParamGrouper

Used to build param_group for torch.optim.Optimizer

from thexp.contrib import ParamGrouper
from thexp import Params

params = Params()
params.optim = params.create_optim('SGD', lr=0.01, momentum=0.9, weight_decay=5e-4)

grouper = ParamGrouper(modela, modelb, ...)
noptim = params.optim.args.copy()
noptim['weight_decay'] = 0
param_groups = [
    grouper.create_param_group(grouper.kernel_params(with_norm=False), **params.optim.args),
    grouper.create_param_group(grouper.bias_params(with_norm=False), **noptim),
    grouper.create_param_group(grouper.norm_params(), **noptim),
]

sgd = params.optim.build(param_groups)

# EMA

EMA(Exponential moving average) is a type of moving average (MA) that places a greater weight and significance on the most recent data points.

thexp provides a function to craete a shadow model for doing EMA operation.

from thexp.contrib import EMA

model = ... # for train
ema_model = EMA(model)

...

optim.step()
ema_model.step() # for eval/test

# Command-Line support

thexp provides a command line tool to support some common used functions.

# reset/archive/delete

# restore code snapshot of some test
thexp reset <test_name>

# archive code snapshot of some test
thexp archive <test_name>

# delete some test directly
thexp delete <test_name>

thexp log <test_name>

# Open Tensorboard by test_name

Tensorboard有时启动会存在临时目录的问题，此外每次都需要定位目录，thexp 额外提供了基于 testname 的定位方式，除此之外，其余参数均和原 tensorboard 命令的参数相同（由于参数解析的问题，该方式不能输入单横杠参数）：

# easier way to open tensorboard 
thexp board [--logdir=<logdir>]
thexp board [--test=<test_name>] # find test_name and tensorboard it 
thexp board  # default open ./board

# Project structure

Here shows the code structure of thexp

  • analyse : For analyzing and visualizing experiments.
  • base_classes : Provided many data structures and basic class.
  • contrib : Expand functions of pytorch
  • calculate : Provided many functions or class for calculating tensor / scalar / space.
  • decorators : All decorators.
  • frame : Whole experiment framework.
  • future : All unstable and unclassified future functions.
  • nest : Expand functions of thexp.
  • util : Basic functions.
