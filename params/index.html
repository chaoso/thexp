# Receive Command-Line Arguments

In most deep learning repositories, the code of receiving command-line arguments is not easy to read (and implement). For pytorch, argparse is an usual choice; for tensorflow, some may use absl.flags. Both of them are not pythonic.

By using Fire, thexp gives a better way to receive command-line arguments.

You can new a file named ap.py, and build a program which can receive arguments by only three line code:



 

from thexp import Params
params = Params()
params.from_args()
1

params.bind('arch','cnn13','arch_param',dict(feature=256))

# Define your own attr

All subclass of attr can be serialized and deserialized (which means that the functions of the subclass can also be used).

from thexp.base_classes import attr
from thexp import BaseParams


class MyAttr(attr):
    def __init__(self) -> None:
        super().__init__()
        self.a = 1
        self.b = 2

    def p(self):
        return 'hello'


a = BaseParams()
a.myattr = MyAttr()
a.to_json('tmp.json')

b = BaseParams()
b.from_json('tmp.json')

assert isinstance(b.myattr, MyAttr)
assert b.p() == 'hello'
